* Wasabi (わさび)

A native Emacs interface for [[https://www.whatsapp.com/][WhatsApp]] messaging.

Powered by [[https://github.com/asternic/wuzapi][wuzapi]] and [[https://github.com/tulir/whatsmeow][whatsmeow]] over standard I/O (json-rpc).

* Setup

**  =wuzapi= dependency

You'll need [[https://github.com/asternic/wuzapi][wuzapi]].

*** Via homebrew

#+begin_src sh
  brew install asternic/wuzapi/wuzapi
#+end_src

*** Build =wuzapi= from source

Requires the [[https://go.dev/][Go]] toolchain.

#+begin_src sh
  git clone https://github.com/asternic/wuzapi.git
  cd wuzapi
  go build .
#+end_src

The compiled =wuzapi= binary will be in the current directory.

You can move it to a location in your PATH or add =wuzapi='s location to Emacs's =exec-path=.

** Install =wasabi=

=Wasabi= isn't on MELPA just yet. You'll need to install it from it's GitHub repo.

You can use something like:

#+begin_src emacs-lisp
  (use-package wasabi
    :vc (:url "https://github.com/xenodium/wasabi"))
#+end_src

/Note: Wasabi currently depends on [[https://github.com/xenodium/acp.el][acp.el]] (available on [[https://melpa.org/#/acp][MELPA]]) for its [[https://www.jsonrpc.org/specification][json-rpc]] implementation and [[https://www.jsonrpc.org/specification][json-rpc]] alone./

* Run

Run with

#+begin_src
  M-x wasabi
#+end_src
